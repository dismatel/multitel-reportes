## Reportar una vulnerabilidad

Para reportar una vulnerabilidad de seguridad en Multitel Reportes:

1. **No abrir un issue público** con detalles de la vulnerabilidad
2. Enviar un email a: jose.reyes@grupomultitel.com
3. Incluir:
   - Descripción del problema
   - Pasos para reproducir
   - Impacto potencial estimado
4. Tiempo de respuesta esperado: 48 horas hábiles

## Alcance

- Azure Functions endpoints
- App Android (React Native)
- Autenticación Azure AD / MSAL
- Datos de campo (GPS, fotos, reportes)

## Lo que NO es una vulnerabilidad

- Bugs de UI o comportamiento inesperado sin impacto de seguridad
- Problemas en dependencias de desarrollo (devDependencies)
